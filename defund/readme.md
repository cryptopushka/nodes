# Defund Node
## Минимальный системные требования 

4 CPU, 8GB RAM, 250GB


## Instal Soft
```
sudo apt install curl -y
```


## Install Node
```
curl -s https://raw.githubusercontent.com/cryptopushka/nodes/main/defund/install.sh > defund_install.sh && chmod +x defund_install.sh && ./defund_install.sh
```
